# ieeks.github.io

Static personal GitHub Pages landing page for `ieeks.github.io`.

## Stack

- HTML
- CSS
- Vanilla JavaScript

## Files

- `index.html` - page structure
- `styles.css` - layout, theme, and responsive styling
- `script.js` - centralized project URLs and theme handling

## Local use

Open `index.html` directly in a browser or serve the folder with any static server.

## Deployment

This repository is GitHub Pages compatible as-is. No build step is required.
